Instalasi Ruby dan Rails di Windows

Berikut adalah langkah-langkah untuk menginstall Ruby dan Rails di Windows:
  • Instalasi Ruby
  • Instalasi Rails

1. Download Ruby Installer for Windows dari tautan berikut: `https://rubyinstaller.org/downloads/` Catatan: Pada instalasi kali ini kita akan menggunakan Ruby+Devkit 2.5.X (x64). Untuk kebutuhan kompilasi gem dengan native extension (C) maka kita membutuhkan MSYS2-Devkit yang sudah diikutsertakan dalam file rubyinstaller tersebut.

2. Jalankan file installer rubyinstaller-devkit-2.5.X-x64.exe, lalu ikuti langkah berikut:

  • Pilih "I Accept the License" lalu klik Next
  • Biarkan setting bawaan dari instalasi (folder, PATH, dll) lalu klik Install
  • Klik Next pada window Select Components
  • Kita akan melakukan setup MSYS2 dan development toolchain dengan ridk lalu klik Finish
  • Ketik 3 pada window setup ruby installer 2 for windows
  • Pastikan instalasi MSYS2 dan MINGW sukses

3. Buka Ruby terminal/command prompt, dengan langkah berikut:

  • Buka Start Menu
  • Klik All Programs
  • Pergi ke Ruby 2.5.X-x64 with MSYS2
  • Buka Start Command Prompt with Ruby

4. Untuk melihat versi Ruby yang sedang digunakan: {% gist ceed7e67ab2ff4cf64b0d2632e10e85c ruby-version.sh %}

Sampai di sini Ruby telah berhasil terinstal, selanjutnya kita akan menginstal Rails. Secara default Gem telah terinstall, perintah di bawah ini untuk memastikan bahwa Kalian menggunakan Gem versi terbaru: {% gist ceed7e67ab2ff4cf64b0d2632e10e85c gem-update-system.sh %}
